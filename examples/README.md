# CEO Skill Examples

These examples help users understand what good outputs look like for different executive decisions.

## Included examples

- `mna-acquisition-decision.md` — acquisition / M&A style decision
- `cto-departure-crisis.md` — crisis response / OODA mode
- `prioritization-five-initiatives.md` — prioritization / ICE scoring mode

## Why these matter

Many users understand a skill faster from examples than from a framework list.
These example files are designed to show:
- when a mode should trigger
- what a strong answer should include
- how structured decision support differs from generic business advice
